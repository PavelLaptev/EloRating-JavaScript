## Elo Rating 
#### Build with CoffeeScript and Compiled in javascript

In this example the data inputed is : rating A: 1400, rating B: 1400

```
Start
---
Everybody starts with 1400 rating
A: 1400 - B: 1400
---
Data inputed: A: 1 - B: 0
---
New Ratings:
---
A: 1506 - B: 1394
---
Now keep changing :) 
```
